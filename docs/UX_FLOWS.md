### Step 2: Configure and Create
**Actions:**
1. (Optional) Edit session name via keyboard
2. Select number of players (default: 3)
3. Select commoners per player (default: 3)
4. Tap "Create Session"

**Result:**
- Session created with auto-generated player names ("Player 1", "Player 2", "Player 3")
- Commoners auto-generated for each player with random names and traits
- Navigate to Players tab automatically

**Time target:** Steps 1-2 = 15 seconds

### 1. One-Tap Advancement

- Names editable by tapping directly

### 3. Quick Actions
- HP: +1, -1, +3, -3 buttons (no keyboard)
- Status: Single tap cycles through states
- Dice: Roll in <3 taps

### 4. Bottom Sheet Overlays
- Commoner details
- Card reveals
- Settings
- 80% screen height max
- Drag to dismiss

### 5. Progress Indicator
- Phase stepper visible at all times during event
- Filled circles for completed phases
- Hollow circle for current
- Labels abbreviated for mobile
